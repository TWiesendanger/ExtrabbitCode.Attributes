---
title: Live Search & Filter
description: Filter the attribute tree by keyword to quickly find the attribute or set you are looking for.
---

The search bar at the top of the Attribute Browser filters the tree in real time as you type. Only nodes that match your query — or that contain a matching child — are shown.

## How filtering works

Matching is case-insensitive and applies to:

- Attribute set names
- Attribute names
- Attribute values

If a child node matches, its parent nodes are kept visible so the result sits in context. Nodes with no match at any level are hidden.

## Clearing the filter

Delete the text in the search bar or click the clear button (×) to restore the full tree.
