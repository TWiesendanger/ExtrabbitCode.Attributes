The Attribute Browser works with all four native Inventor document types. Switch between open documents and the panel updates to reflect the attributes of whatever file is currently active (when [Auto-Refresh](/docs/features/auto-refresh) is enabled).

## Assemblies and components

When you have an assembly open, the Attribute Browser shows attributes that belong to the **assembly document itself**. To inspect attributes on an individual component, open or activate that component's document in Inventor — the panel updates automatically.
