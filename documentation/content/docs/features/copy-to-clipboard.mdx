title: Copy to Clipboard
description: Copy any attribute value directly from the tree for use in other tools.
---

Any attribute value in the tree can be copied to the clipboard without opening an edit dialog.

{/* Screenshot: right-click context menu on an attribute node with "Copy value" highlighted */}

## How to copy a value

Select an attribute node in the tree and use the **Copy** action (toolbar button or right-click context menu). The attribute's value is placed on the system clipboard and can be pasted anywhere — a spreadsheet, a note, another application.

## What is copied

Only the **value** of the selected attribute is copied — not the attribute name or set name.
